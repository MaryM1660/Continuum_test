# Применение официальных гайдлайнов Apple HIG ко всему приложению

## Цвета по Apple HIG

### Системные цвета iOS 17/18:
- **System Blue**: #007AFF (iOS стандартный синий)
- **System Gray**: #8E8E93 (вторичный текст)
- **System Background**: #FFFFFF (светлая тема), #000000 (темная тема)
- **System Grouped Background**: #F2F2F7 (светлая), #000000 (темная)

### Текст:
- **Label**: #000000 (светлая), #FFFFFF (темная)
- **Secondary Label**: #3C3C43 (светлая, 60% opacity), #EBEBF5 (темная, 60% opacity)
- **Tertiary Label**: #3C3C43 (светлая, 30% opacity), #EBEBF5 (темная, 30% opacity)

## Типографика SF Pro

### Размеры шрифтов (iOS):
- **Large Title**: 34pt, weight 700
- **Title 1**: 28pt, weight 700
- **Title 2**: 22pt, weight 700
- **Title 3**: 20pt, weight 600
- **Headline**: 17pt, weight 600
- **Body**: 17pt, weight 400
- **Callout**: 16pt, weight 400
- **Subhead**: 15pt, weight 400
- **Footnote**: 13pt, weight 400
- **Caption 1**: 12pt, weight 400
- **Caption 2**: 11pt, weight 400

### Letter Spacing:
- Заголовки: 0.37-0.36
- Body: -0.41
- Мелкий текст: -0.08 до 0

## Spacing (8pt grid для iOS):
- 4pt, 8pt, 12pt, 16pt, 20pt, 24pt, 32pt, 40pt, 48pt, 64pt

## Liquid Glass эффект:
- `backdrop-filter: blur(20px)`
- `background: rgba(255, 255, 255, 0.7)` (светлая), `rgba(0, 0, 0, 0.7)` (темная)
- `border: 1px solid rgba(255, 255, 255, 0.18)`
- `box-shadow: 0 8px 32px rgba(0, 0, 0, 0.12)`
