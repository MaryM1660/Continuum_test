## Примечания

- Размеры текста и элементов управления различаются для разных платформ
- Минимальные размеры обеспечивают доступность для пользователей с ограниченными возможностями
- Контрастность текста зависит от размера и веса шрифта
- tvOS имеет самые большие размеры по умолчанию (29 pt для текста, 66x66 pt для элементов управления)
- macOS имеет самые маленькие размеры по умолчанию (13 pt для текста, 28x28 pt для элементов управления)
