## Важное замечание о единицах измерения

### React Native на веб
В React Native для веб:
- `fontSize: 17` в коде = **17px** в браузере
- Это **правильно** и соответствует стандартам

### React Native на мобильных
На iOS/Android:
- `fontSize: 17` конвертируется в логические пиксели
- На iOS: 17 логических пикселей ≈ 17pt (на Retina дисплеях)
- На Android: 17 логических пикселей = 17sp (scalable pixels)

### Вывод
✅ **Размеры указаны правильно** - они будут корректно отображаться и на веб, и на мобильных устройствах.
